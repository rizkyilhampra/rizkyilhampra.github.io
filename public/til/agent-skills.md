A hub for everything related to **SKILL.md packages** — the agent-skill manifest spec that teaches AI agents specialized workflows, taste, and domain expertise.

## What is a skill?

A `SKILL.md` file is a manifest (name + description + instructions) that an agent loads on-demand to gain focused capability for a task.
